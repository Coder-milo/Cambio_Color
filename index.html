Lenguejes de programacion

Son sistemas de reglas que permiten escribir instrucciones para que las computadoras realicen tareas.

Tipos de lenguajes

C++: Lenguaje de alto rendimiento de programación orientada a objetos y uso eficiente de memoria, usado en sistemas, videojuegos, aplicaciones.
Python: Lenguaje de propósito general con sintaxis sencilla y clara, ideal para desarrollo web, ciencia de datos, automatización y más.
C#: Lenguaje desarrollado por Microsoft, muy usado en aplicaciones empresariales y videojuegos con Unity.
SQL: Lenguaje utilizado para gestionar y consultar bases de datos relacionales.
PHP: Lenguaje de servidor usado para desarrollar sitios web dinámicos e interactuar con bases de datos.
Scala: Combina programación funcional y orientada a objetos, usado en aplicaciones escalables y en el procesamiento de datos con Apache Spark.
